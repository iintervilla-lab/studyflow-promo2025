# Instructions de déploiement StudyFlow

## 1. Créer un repo GitHub
- Allez sur https://github.com/new
- Nom du repo : `studyflow-promo2025`
- Description : "Espace collaboratif pour la Promo 2025"
- **Rendre PUBLIC** (pour que tes amis puissent y accéder)
- Ne pas cocher "Add a README" (on en a déjà un)

## 2. Pousser le code
Après avoir créé le repo, exécutez ces commandes :

```bash
# Ajouter l'origine GitHub (remplacez VOTRE_USERNAME par votre nom GitHub)
git remote add origin https://github.com/VOTRE_USERNAME/studyflow-promo2025.git

# Pousser le code
git push -u origin master
```

## 3. Activer GitHub Pages
- Dans le repo GitHub, allez dans Settings > Pages
- Source : "Deploy from a branch"
- Branch : master
- Folder : /(root)
- Save

## 4. URL de l'app
L'URL sera : https://VOTRE_USERNAME.github.io/studyflow-promo2025

Partage cette URL à tes amis ! 📱💬

## 5. Installation sur téléphone
- Ouvrez l'URL sur votre téléphone
- Appuyez sur "Ajouter à l'écran d'accueil"
- L'app sera installée comme une vraie appli mobile !
