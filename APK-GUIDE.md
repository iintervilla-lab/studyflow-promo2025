## 📱 Méthodes de Génération APK

### 🥇 **Méthode 1: PWABuilder (Recommandée)**
1. **Ouvrez** : https://www.pwabuilder.com/
2. **Collez l'URL** : `https://iintervilla-lab.github.io/studyflow-promo2025`
3. **Cliquez** : "Start"
4. **Sélectionnez** : "Android"
5. **Cliquez** : "Download"

### 🥈 **Méthode 2: Bubblewrap (CLI)**
```bash
# Installer Bubblewrap
npm install -g @bubblewrap/cli

# Initialiser avec le manifest
bubblewrap init --manifest https://iintervilla-lab.github.io/studyflow-promo2025/manifest.json

# Générer l'APK
bubblewrap build
```

### 🥉 **Méthode 3: PWA2APK (En ligne)**
1. **Allez sur** : https://www.pwa2apk.com/
2. **Entrez l'URL** : `https://iintervilla-lab.github.io/studyflow-promo2025`
3. **Générez l'APK**

## 📦 Caractéristiques de l'APK

✅ **Mode hors ligne complet** - Fonctionne sans connexion internet
✅ **Synchronisation automatique** - Données synchronisées quand connexion revient
✅ **Interface native Android** - Raccourcis, icônes, comportement natif
✅ **Notifications push** - Alertes de synchronisation et rappels
✅ **Performance optimisée** - Cache intelligent et chargement rapide

## 📋 Instructions d'Installation

1. **Transférez** le fichier `.apk` sur votre téléphone Android
2. **Paramètres** > **Sécurité** > Autoriser les sources inconnues
3. **Ouvrez** le fichier `.apk` et installez l'application
4. **Lancez** StudyFlow depuis votre écran d'accueil

## 🔧 Dépannage

- **APK ne s'installe pas** : Vérifiez les permissions de sécurité
- **Mode hors ligne ne marche pas** : Videz le cache de l'app
- **Synchronisation lente** : Vérifiez votre connexion réseau

## 📞 Support

Si vous rencontrez des problèmes :
1. Testez l'app web d'abord : https://iintervilla-lab.github.io/studyflow-promo2025
2. Vérifiez la console du navigateur (F12)
3. Contactez le développeur avec les détails de l'erreur
